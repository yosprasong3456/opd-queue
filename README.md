# OPD Queue 

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

## Setup Project
Create .env file

```sh
REACT_APP_API_URL =
REACT_APP_API_TESTSITE=http://localhost/opd_queue_api/
REACT_APP_MY_ENVIRONMENT_VARIABLE=test
```
